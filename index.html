HTML Basics

HTML stands for HyperText Markup Language. It is the standard language used to create web pages.

<html>

HTML is made up of elements, which are represented by tags.

<head>

The head section contains meta information, links to CSS files, and JavaScript files.

<body>

  • <footer>
  • <main>
  • <section>
  • <article>

CSS Styling

CSS stands for Cascading Style Sheets. It is used to style HTML elements on a webpage.

body { background-color: lightblue; }

CSS rules consist of selectors and declaration blocks.

h1 { color: blue; }

CSS can be added to HTML in three ways: inline, internal, and external.

  • Internal styles
  • External styles

NodeJS Basics

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine.

const http = require('http');

It allows JavaScript to be run on the server side.

http.createServer((req, res) => { res.write('Hello World'); res.end(); }).listen(8080);
